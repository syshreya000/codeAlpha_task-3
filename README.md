#  Personal Portfolio Website
A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript. This project showcases my skills, projects, and contact information with a clean and user-friendly interface.

##  Features
- Responsive design for desktop and mobile
- Modern and attractive UI
- Smooth navigation
- About Me section
- Skills showcase
- Projects section
- Resume download button
- GitHub and LinkedIn links
- Scroll animations

##  Technologies Used
- HTML5
- CSS3
- JavaScript
